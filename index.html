PROJECTS

HAWK.AI

Github Repo: https://github.com/AshlinJoelSizzin/HawkAI

Domain: Full Stack Development and Human Computer Interaction using AI, ML & DL

TECH STACK USED: HTML, CSS, Advanced CSS, Javascript, Bootstrap, NodeJS, EJS, Express, PostgreSQL, RESTFUL APIs, Media Pipe, Open CV.

HawkAI is an innovative smart restaurant food ordering system that leverages Human-Computer Interaction (HCI) principles to provide a seamless and inclusive dining experience. This system is designed to address the increasing demand for contactless ordering solutions, particularly benefiting individuals with disabilities. HawkAI utilizes real-time gesture recognition technology, allowing customers to navigate the restaurant menu and place orders through unique gestures, reducing the need for physical interaction. The web application features a comprehensive menu interface and integrates a real-time camera for gesture-based navigation. Built using HTML, CSS, JavaScript, EJS Templating, Node.js, Express.js, and PostgreSQL, HawkAI also incorporates RESTful APIs and is in the process of integrating a gesture recognition algorithm using Flask. As the project lead, I conceptualized the idea, oversaw the web development, and designed the user interface to ensure it aligns aesthetically with a restaurant environment. My team is currently finalizing the gesture recognition algorithm for full integration into the system.

ROCKFETCH

Github Repo: https://github.com/AshlinJoelSizzin/RockFetch

Domain: Full Stack Development

TECH STACK USED: HTML, CSS, Advanced CSS, Javascript, NodeJS, EJS, Express, PostgreSQL, RESTFUL APIs, Multer, Salting and Hashing

RockFetch is a comprehensive web application designed to address the essential needs of hostel students, offering three core functionalities: Store, Carpooling, and Networking. The Store feature facilitates the buying and selling of pre-owned items among hostel students, providing a convenient marketplace for exchanging goods within the hostel community. Carpooling allows users to post travel plans and share rides, promoting cost-sharing and reducing the challenge of traveling alone. The Networking platform serves as a collaborative space where like-minded individuals can share ideas and work on projects together. This web application is built using modern technologies, including HTML, CSS, JavaScript, EJS Templating, Node.js, Express.js, and PostgreSQL. It also employs RESTful APIs for seamless interaction between the frontend and backend, and Multer for efficient file handling. Security is ensured through salting and hashing techniques. The branding, including the logo, was designed in-house, with the user interface tailored to be interactive and engaging for the student demographic. This is a group project where I played a key role, originating the concept and executing the majority of the development work.

Enhanced Human-Computer Interaction for Smart Surgical Systems: Empowering Surgeons with Precision Control in Operating Theatres

TECH STACK USED: Python, Human Computer Interaction, Gesture Recognition, Open CV, Media Pipe, Tensorflow, Socket Library, IoT

This project is an AI, ML, and IoT-driven surgical automation system. In this project, I identified a significant inefficiency in traditional surgical procedures where multiple professionals are required to manually configure and operate various devices such as lights, ultrasound machines, and surgical cameras. The goal is to develop a system that consolidates the control of these devices into a single interface, enabling a surgeon to adjust settings such as light intensity, ultrasound wavelength, and camera focus with minimal assistance. This is achieved by designing an algorithm that uses human-computer interaction to communicate with IoT-enabled devices via a digital-to-analog converter, optimizing surgical procedures by reducing time and effort while increasing precision.

ThermoSense: AI-Powered Physiological Monitoring and Risk Assessment System

TECH STACK USED: Python, Human Computer Interaction, Gesture Recognition, Open CV, Media Pipe, Tensorflow, Socket Library, IoT, Temporal Convolution Networks, Gated Recurrent Unit.

This project involves human health risk assessment using AI, ML, and IoT technologies. This project aims to monitor a person's physical activity and body temperature through IoT sensors in devices like smartphones or smartwatches. The algorithm being developed will detect when a person is engaging in strenuous physical activity and alert them to rest once a predefined threshold of body temperature and stress is reached. To enhance the accuracy and responsiveness of the system, Temporal Convolutional Networks (TCN) and Gated Recurrent Units (GRU) are being utilized in the algorithm. TCNs are effective for capturing long-range dependencies and processing sequential data, while GRUs are known for their efficiency in handling time series data, particularly in real-time monitoring scenarios. Together, these models improve the system's ability to analyze patterns in physical activity and temperature fluctuations, providing timely and accurate feedback to help individuals avoid potential health risks and promote long-term well-being.
